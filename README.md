# 中山大学 LMS 视频自动播放器



一个用于 \*\*中山大学在线教学平台（LMS）\*\* 的 Userscript 脚本。  

它会在课程视频页面自动完成以下流程：



- 自动等待视频加载

- 自动开始播放

- 自动设置播放倍速

- 可选跳过已完成（100%）的视频

- 当前视频播放结束后自动跳转到下一个视频

- 遇到加载失败时自动重试

- 视频长时间无进度时自动跳过

- 在页面右下角显示当前运行状态


适合用来减少重复手动点播切换视频的操作。



---



## 功能特性



### 1. 自动播放视频

脚本会等待页面中的视频播放器加载完成，然后自动尝试开始播放。



### 2. 自动设置倍速

支持在配置中设置播放速度，例如：



- `1`：正常速度

- `1.5`：1.5 倍速

- `2`：2 倍速



> 实际可用倍速取决于 LMS 页面和浏览器策略。



### 3. 自动跳过已完成视频

当检测到当前视频进度已经达到 `100%` 时，脚本可直接跳到下一个视频，避免重复播放。



### 4. 自动跳转到下一个视频

当前视频结束后，脚本会自动寻找下一个视频地址并跳转。



优先使用：



- 页面中的视频选择下拉框

- 页面中的“下一个活动”链接



### 5. 异常恢复与重试

如果播放器迟迟没有加载出来，脚本会在一定次数内自动重试。



### 6. 卡住检测

脚本会定时检查视频播放进度：



- 若视频长时间没有推进

- 且持续达到设定阈值



则会判断该视频可能已卡住，并自动跳转到下一个视频。



### 7. 页面状态浮窗

脚本会在页面右下角显示状态提示，例如：



- 正在初始化

- 等待播放器加载

- 正在播放

- 缓冲中

- 已完成，准备跳转

- 所有视频播放完毕
